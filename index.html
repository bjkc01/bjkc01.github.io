Project 05 — Security Program Lead (Intern Project)

Cybersecurity Awareness & Phishing Simulation Program

Designed and deployed a security awareness training program leveraging Huntress to address phishing, social engineering, and user-level attack vectors across the organization. Engineered and executed simulated phishing campaigns using real-world attack scenarios (credential harvesting, malicious links, spoofed domains) to assess human attack surface risk. Applied threat modeling and adversarial thinking to continuously refine phishing templates, aligning simulations with MITRE ATT&CK techniques (T1566 – Phishing).
